# SonarInterestPointDetection

## Installation
### Requirements
- Python 3 >= 3.5
- PyTorch >= 1.1
- OpenCV >= 3.4
- NumPy >= 1.18
```
conda crreate --name sonar-ip python=3.6
conda activate sonar-ip
pip install -r requirements.txt
```
